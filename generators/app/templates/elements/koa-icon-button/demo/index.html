Buttons can use iron-icons or external images, and can be disabled

<<%= tagName %> icon="favorite" title="heart">> <<%= tagName %> src="https://assets-cdn.github.com/images/modules/logos_page/Octocat.png" alt="octocat" title="octocat">> <<%= tagName %> disabled icon="reply" title="reply">>

Buttons can be styled using regular CSS and custom properties

<<%= tagName %> icon="favorite" title="heart" class="pink">> <<%= tagName %> icon="flight-takeoff" title="take off" class="blue">>

Buttons can be resized

<<%= tagName %> icon="alarm-on" title="wake up" class="giant">>
